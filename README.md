# c# runner for !  `</vscode>`
linux users should install `wine32`

## Setup

#### for windows
```bat
git clone "https://github.com/CmdErr0r/run-cs"
cmd /c "run-sh\c#.bat"
```

#### for linux
```sh
git clone "https://github.com/CmdErr0r/run-cs"
wine cmd /c "run-sh\c#.bat"
```

## Run
```sh
# for debug press Fn + F5 on vscode #
cd Path/To/Directory/
cd TemplateName
dotnet run
```

@dev stop
